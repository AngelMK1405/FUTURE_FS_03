# FUTURE_FS_03
# Apex Fitness Club Website | Future Interns Task 3

## Project Overview
This project is my submission for **Task 3** of the Full Stack Web Development Internship at Future Interns.

The task required building a professional website for a real local business and creating a pitch explaining how the website helps the business grow.

For this project, I selected **Apex Fitness Club**, a hardcore bodybuilding gym located in Kottayam, Kerala.

---

## Business Problem
Many local gyms rely heavily on walk-in customers and word-of-mouth marketing, with little to no professional online presence.

This creates several challenges:
- Limited visibility
- Difficulty attracting new customers
- Lack of credibility
- Poor online engagement

---

## Solution
I designed and developed a modern, responsive business website for Apex Fitness Club.

The website provides:
- Professional branding
- Service showcase
- Membership plans
- Trainer profiles
- Contact and inquiry system
- Google Maps location access

---

## Features
- Fully responsive design
- Sticky navigation bar
- Dark mode toggle
- Animated statistics counters
- Membership pricing toggle
- Transformation gallery
- Testimonials
- WhatsApp contact button
- Formspree-powered contact form
- Google Maps integration
